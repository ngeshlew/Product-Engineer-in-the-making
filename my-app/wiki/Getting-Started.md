# Getting Started

## Clone
```bash
git clone <your-new-repo>
cd <repo>
```

## Initialize
```bash
bash scripts/init.sh
```

Open in Cursor and start with `/init` then `/plan`.

## Commands
- /init, /plan, /execute, /review, /commit

## Optional
- Configure `.cursor/mcp.json` and export tokens for GitHub/Brave if needed
